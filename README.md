# Routine Managaer

## Funktionen

- Eine Routine bestehende aus Bild der Leistung, zeit, Attribute der akktivität bspw. gelaufene km, verbrante kalorien, lern erfolg, Motivation
- Statistiken Fenster mit tage in folge, minuten bzw zeit vebracht, gelernte sachen falls vorhanden
- Radar-Chart (Lebensbereiche) Zeit-Verteilung pro Woche Vergleich: letzte Woche vs. diese Woche, Produktivitäts-Score hand in hand mit statistiken wenn man bspq runter scrolled
- wenn etwas großes neues gelernt 5 min reminder video wie was funktioniert für spätere wiederaufarbeitung
- ziele setzen 
- Forecast-Modell Wenn du so weitermachst → Ziel erreicht am X
- profil erstellen
- Erinnerungen Custom
- Zeit pro Kategorie (Heatmap wie GitHub Contributions)
- inform von einem kommentar Selbstbewertung nach jeder Lernsession was gut schlecht war
- Analyse-Dashboard (wichtiger Punkt für GitHub)

## funktionen für zukunft und ausarbeitung

- communitys/gruppen
- Konsistenz-Score bestehende Gewichtung aus: Streak, Erfüllungsquote, Intensität
